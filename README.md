# AI Hackathon FRA Slides

A Slidev project using the `slidev-theme-scholarly` theme.

## Use

```bash
npm install
npm run dev
```

## GitHub Pages

This repository includes a GitHub Actions workflow at `.github/workflows/deploy-pages.yml`.

After pushing to the `main` branch, enable GitHub Pages in the repository settings:

- Source: GitHub Actions

The workflow builds `slidev.md` and publishes the static site automatically.
